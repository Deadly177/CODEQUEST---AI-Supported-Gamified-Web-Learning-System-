# CodeQuest Testing

This folder stores testing work for the CodeQuest project.

## Folder Structure

- `reports/` - Short testing reports written after each test activity.
- `results/` - Raw test outputs, logs, screenshots, or exported tool results.

## Report Naming

Use this format:

```text
reports/01-test-name-report.md
results/01-test-name-results.txt
```

Example:

```text
reports/01-backend-white-box-progress-report.md
results/01-backend-white-box-progress-results.txt
```
