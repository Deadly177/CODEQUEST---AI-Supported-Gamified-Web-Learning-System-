Usability Testing Protocol - CodeQuest

Status: Prepared for human participant testing.

Recommended Participants:
- 3 to 5 beginner programming learners.

Tasks:

Task 1: Register a new account.
Success Criteria: Participant completes registration without help.

Task 2: Find the HTML or JavaScript course.
Success Criteria: Participant opens the correct course page.

Task 3: Start the first available lesson.
Success Criteria: Participant opens the lesson player and understands what to do next.

Task 4: Complete one lesson interaction or quiz step.
Success Criteria: Participant completes the step and sees progress feedback.

Task 5: Use the Build Workspace to change the heading text.
Success Criteria: Participant edits HTML and sees the preview update.

Task 6: Ask the AI Assistant a learning question.
Success Criteria: Participant sends a question and understands the returned result or error state.

Suggested Metrics:
- Task success: success/fail
- Time taken per task
- Number of mistakes
- Number of times help was requested
- Participant satisfaction rating from 1 to 5
